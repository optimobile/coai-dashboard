## Analyst Assignment System

### How It Works:

1. **Analyst Registration** → User selects region (EU/US/UK/etc.)
2. **Regional Routing** → User sees region-specific courses
3. **Training** → User completes regional compliance bundle
4. **Certification** → User passes region-specific exam
5. **Specialist Assignment** → Regional specialist assigns analyst to industry specialist
6. **Work Assignment** → Industry specialist assigns watchdog reports to analyst
7. **Quality Review** → Specialist reviews analyst work, provides feedback
8. **Performance Tracking** → Analyst builds track record under specialist supervision
